PERSONAL PROFILE

Personal Profile

According to the Myers—Briggs test, I’m Logistician. The characteristics of this personality are honest and direct, strong-willed and dutiful, very responsible, calm and practical, create and enforce order, jacks-of-all trades. According to the website, this personality does everything to get work done no matter what. Even though this type of person work well individually but also can work in a team. These persons tend to be stubborn, stick with the theory and findings. When working in a team, this personality will tend to do everything, get work done before asking. In order to achieve goals, constant improvement is needed. My learning style is tactile, which is learning through touching and doing. I believe this is the way to learn IT especially with programming. To learn a new coding language, a lot of practices are need. This will help to learn the characteristics of the function, understand the code, understand bugs as well as advantages and disadvantages of the language. According to Adobe creativity test, I’m a Maker. Maker is usually one of the hardest workers, busy solving problems. Not only that, focus and dedication are the main advantages of this personality. With all these characteristics, I think I will be attractive to employers since everyone will want a hard-working teammate in the group. With the learning style and Maker personality, it will help a lot in learning and practicing IT professionally.
